```uml
@startuml
[*] -> トップページ
トップページ -> 商品詳細 : 商品をクリック
商品詳細 -> トップページ : トップページをクリック
商品詳細 : do/絵本の詳細を表示
トップページ --> 会員情報 : 会員情報をクリック
会員情報 -> トップページ : トップページをクリック
トップページ -left-> ログイン画面 : ログインをクリック
ログイン画面 -left-> トップページ : トップページをクリック
ログイン画面 : do/ログイン認証を行うページ
トップページ --> カート : カートをクリック
カート --> トップページ : トップページをクリック
商品詳細 --> カート : カートに入れるをクリック
カート --> 商品詳細 : 商品をクリック
カート : do/カートに入れた商品を表示
トップページ --> おすすめ絵本 : おすすめをクリック
おすすめ --> トップページ : トップページをクリック
おすすめ --> 商品詳細 : 商品をクリック
商品詳細 --> おすすめ : 戻るボタンをクリック
おすすめ : do/おすすめの絵本を表示する
トップページ --> 検索結果 : 検索ボタンをクリック
検索結果 --> トップページ : トップページをクリック
検索結果 : do/キーワード検索の結果を表示する
トップページ ---> 問い合わせページ : 問い合わせをクリック
問合せページ ---> トップページ : トップページをクリック
トップページ ---> 新着絵本 : 新着絵本をクリック
新着絵本 ---> トップページ : トップページをクリック
@enduml
```
